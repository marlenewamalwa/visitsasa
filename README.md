Visit Sasa
Visit Sasa is a web platform for discovering destinations, attractions, and local experiences in Kenya. It focuses on simple navigation and a responsive user interface to promote travel and cultural exploration.

Tech Stack
HTML, CSS, JavaScript, PHP, MySQL

Features
Browse destinations and attractions
Responsive and user-friendly design
Easy content expansion

Setup
